Please deploy your application on CS3, and package your project in a zip file and submit it to Canvas. The source files should include all the source code, and an HTML file final.html which contains a hyperlink to your application on the CS3 server.

In this exam you are going to develop a review manager web application that is similar to the ones that can be found on many smart phones.

Your application must use a database to store the data, and you must submit an SQL script file final.sql which contains the statements to create and populate all the tables for the application. Your implementation must follow the MVC architecture and use JDBC for database access. For simplicity, we assume that all user input are correct so you do not need to do input validation.

The main page of the application displays the names of the attractions managed by the application.

After an attraction is added, the user should be redirected to the attractions list page. Clicking on Attractions also takes the user back to the attraction list page.

Clicking on the name of an attraction in the attractions list allows the user to see all the reviews about the attraction as well as adding more reviews to the attraction:

Note that after a new review is added, the user should be redirected back to the attractions page, and there is no limit on how many reviews a user can add to an attraction.
